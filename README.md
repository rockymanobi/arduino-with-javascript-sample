
## Get Started

### 前提

* ArduinoがUSB接続されている
* Firmataライブラリがインストールがインストールされている

### 実行

#### with GIT

```
git clone http://github.com/rockymanobi/arduino-with-javascript-sample.git
cd arduino-with-javascript-sample
npm install
node sample
```

#### without GIT

zipでダウンロード `URLURLURLURL`

```
cd ダウンロードしたディレクトリ
cd arduino-with-javascript-sample
npm install
node sample
```
